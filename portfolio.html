Purpose

This portfolio website is designed to showcase my work, skills, and achievements to potential clients or employers in a unique and engaging manner. Its structure is built with expandability in mind, with elements and pages styled and organized to accommodate future growth. The website serves as a powerful tool to present my portfolio and highlight my capabilities to potential clients or employers.

Features

Image Carousel

The image carousel on the website was built from scratch and is used on the homepage to showcase all of my certificates. It is a useful way to present a large number of images in a compact space and allows users to easily browse through them.

Vanilla Javascript

Every animated or interactive element on the website has some form of javascript, all of which is implemented without the use of jQuery or any animation library. This provides a lightweight and efficient solution for adding interactive features to the website.

Responsive Images

The website uses responsive images to ensure that all images are displayed at the optimal size and resolution for the device they are being viewed on. To achieve this, the images have a wide range of sizes and formats, allowing the browser to choose the most appropriate one to display. This ensures that the images look sharp and clear on all devices, including smartphones, tablets, and desktop computers.

Accessibility

Accessibility has been a key consideration in the design and development of this website. The entire website can be navigated using only keyboard inputs, and aria labels are used wherever they are needed to provide context and meaning for screen readers. Alt attributes are provided for all images to describe their content for users with visual impairments, and every interactive element is big enough to ensure that it is easy to use for users with motor control impairments. These features make the website more accessible and easier to use for users with disabilities.

Spanish Translation

The website includes a Spanish translation, which can be accessed through a dedicated subdomain. This allows Spanish-speaking visitors to access the content in their preferred language, with a language selection box at the bottom of the page providing an easy way to switch between languages. This is a useful feature for websites that target a bilingual or multilingual audience.
